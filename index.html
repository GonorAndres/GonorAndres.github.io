Proyectos Recreativos

  • Uso de Simulación Montecarlo para Entender el "Azar" del Poker

    Los juegos de cartas, como el póker, ofrecen un entorno ideal para introducir conceptos de simulación. Este proyecto utiliza la técnica de Montecarlo para estimar probabilidades específicas y comprender el comportamiento del azar. El cuaderno fue redactado en inglés, como parte de una práctica profesional.

    Github Proyecto Texas Poker Case Study

  • Comparación entre Inferencia Bayesiana y Frecuentista en un Test A/B

    En este proyecto se aborda un problema clásico de pruebas A/B: evaluar si una nueva versión mejora la tasa de conversión. Se aplican tanto métodos frecuentistas como técnicas bayesianas, incluyendo modelado con PyMC y distribución Beta. El análisis está desarrollado en inglés, acompañado de un resumen explicativo en español.

    Github Proyecto AB Testing Bayes vs Frequentist

Proyectos Escolares

  • Análisis Demográfico del estado de Michoacán

    Usando información de INEGI del censo 2020, se estudió la composición de la población, obteniendo indicadores como la mortalidad, la fecundidad y la estructura por edades. Elaborado con Excel y conocimientos del curso de demografía.

    Drive Proyecto Análisis Michoacán de Ocampo

  • Limpieza de Base de Datos – Deuda Pública CDMX

    Proyecto de limpieza de una base de deuda pública del Gobierno de la Ciudad de México. El trabajo se realizó en Microsoft Excel, aplicando funciones como BUSCARV(), SI() y verificación de fechas. Se reconstruyó la tasa TIIE y se obtuvo una base lista para análisis estadístico.

    Drive Proyecto Limpieza de Datos – Deuda Pública

    Proyecto práctico del curso de Matemáticas Financieras. Se desarrolló un amortizador en Excel basado en parámetros definidos por el usuario como monto, tasa y tipo de pago. Incluye respuestas al examen y PDF anexo.

  • Mercado de Derivados de Divisas – MCF Proyecto 1

    Proyecto desarrollado en el curso de Métodos Cuantitativos en Finanzas, enfocado en el análisis del mercado de divisas. Incluye la construcción de la curva forward, cálculo de precios forwards y evaluación de una operación concreta con spread bid/offer. Se complementa con una superficie de volatilidad e interpolación de precios de opciones Call y Put. Desarrollado en Python.

    GitHub Proyecto Mercado de Derivados de Divisas

  • Modelo Predictivo de Incumplimiento Crediticio

    Desarrollo de un modelo de machine learning con 85.19% de precisión (AUC = 0.8519) para predecir incumplimiento en préstamos. Utilizando técnicas de GLM en Python, se identificaron 5 factores clave de riesgo y se procesó un dataset con más de 32,000 registros. El proyecto incluye controles de calidad, validación cruzada y documentación completa en inglés con enfoque en transparencia regulatoria.

    Github Proyecto Credit_Risk;Model

  • Optimización de Portafolio de Inversión: Teoría de Markowitz

    Aplicación práctica de la teoría moderna de portafolios con 10 activos del mercado mexicano, logrando un 110% de rendimiento anualizado en período de prueba. Se implementó optimización cuadrática para minimización de riesgo con restricciones de diversificación usando Excel y métodos cuantitativos. Incluye análisis de correlaciones, cálculo de VaR paramétrico y pruebas de estrés con backtesting.

    Drive Proyecto Portfolio Optimization Markowitz

  • Método de Euler para Ecuaciones Diferenciales Ordinarias

    Implementación completa del Método de Euler para resolver EDOs de primer orden, aplicado al problema $x'(t) = 1 + (x-t)^2$ con $x(0) = 1/2$. El proyecto incluye demostración de existencia y unicidad usando el teorema de Picard-Lindelöf, construcción de funciones lineales por tramos y análisis de convergencia comparando con la solución analítica $x(t) = t + \frac{1}{2-t}$. Desarrollado en Python con visualizaciones usando matplotlib.

    Google Colab Método de Euler - EDOs
