Proyectos Recreativos

  • Uso de Simulacion Montecarlo para Entender el "Azar" del Poker

    Los juegos de cartas, como el poker, ofrecen un entorno ideal para introducir conceptos de simulacion. Este proyecto utiliza la tecnica de Montecarlo para estimar probabilidades especificas y comprender el comportamiento del azar. El cuaderno fue redactado en ingles, como parte de una practica profesional.

    Github Texas Poker Case Study

  • Comparacion entre Inferencia Bayesiana y Frecuentista en un Test A/B

    En este proyecto se aborda un problema clasico de pruebas A/B: evaluar si una nueva version mejora la tasa de conversion. Se aplican tanto metodos frecuentistas como tecnicas bayesianas, incluyendo modelado con PyMC y distribucion Beta. El analisis esta desarrollado en ingles, acompanado de un resumen explicativo en espanol.

    Github AB Testing Bayes vs Frequentist

  • Modelo Predictivo de Incumplimiento Crediticio

    Desarrollo de un modelo de machine learning con 85.19% de precision (AUC = 0.8519) para predecir incumplimiento en prestamos. Utilizando tecnicas de GLM en Python, se identificaron 5 factores clave de riesgo y se proceso un dataset con mas de 32,000 registros. El proyecto incluye controles de calidad, validacion cruzada y documentacion completa en ingles con enfoque en transparencia regulatoria.

    Github Credit Risk Model

  • Limpieza de Base de Datos - Deuda Publica CDMX

  • Mercado de Derivados de Divisas - MCF Proyecto 1

    Proyecto desarrollado en el curso de Metodos Cuantitativos en Finanzas, enfocado en el analisis del mercado de divisas. Incluye la construccion de la curva forward, calculo de precios forwards y evaluacion de una operacion concreta con spread bid/offer. Se complementa con una superficie de volatilidad e interpolacion de precios de opciones Call y Put. Desarrollado en Python.

    GitHub Mercado de Derivados de Divisas

  • Optimizacion de Portafolio de Inversion: Teoria de Markowitz

    Aplicacion practica de la teoria moderna de portafolios con 10 activos del mercado mexicano, logrando un 110% de rendimiento anualizado en periodo de prueba. Se implemento optimizacion cuadratica para minimizacion de riesgo con restricciones de diversificacion usando Excel y metodos cuantitativos. Incluye analisis de correlaciones, calculo de VaR parametrico y pruebas de estres con backtesting.

    Drive Portfolio Optimization Markowitz

  • Metodo de Euler para Ecuaciones Diferenciales Ordinarias

    Implementacion completa del Metodo de Euler para resolver EDOs de primer orden, aplicado al problema x'(t) = 1 + (x-t)^2 con x(0) = 1/2. El proyecto incluye demostracion de existencia y unicidad usando el teorema de Picard-Lindelof, construccion de funciones lineales por tramos y analisis de convergencia comparando con la solucion analitica x(t) = t + 1/(2-t). Desarrollado en Python con visualizaciones usando matplotlib.

    Google Colab Metodo de Euler - EDOs
